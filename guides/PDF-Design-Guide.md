# PDF Design Guide for Benjamin Mlynek Guides

This guide will help you create professional PDF documents that match your website's branding and aesthetic.

## Typography

### Primary Font
- **Font Family**: Inter (Google Fonts)
- **Font Weights**: 400 (Regular), 500 (Medium), 600 (Semibold), 700 (Bold)
- **Download**: https://fonts.google.com/specimen/Inter

### Font Hierarchy
1. **Main Heading (H1)**: Inter Bold, 32-36pt, Primary Orange
2. **Section Heading (H2)**: Inter Semibold, 24-28pt, Foreground Light
3. **Subsection (H3)**: Inter Semibold, 18-20pt, Foreground Light
4. **Body Text**: Inter Regular, 11-12pt, Foreground Light
5. **Caption/Small**: Inter Regular, 9-10pt, Muted Text

## Layout Guidelines

### Page Setup
- **Page Size**: A4 (210 √ó 297 mm)
- **Margins**: 20mm all sides
- **Content Area**: 170 √ó 257 mm

### Grid System
- **Columns**: 2-column layout for content sections
- **Gutter**: 10mm between columns
- **Line Height**: 1.5x font size for body text
- **Paragraph Spacing**: 6-8pt after paragraphs

### Header Design
- **Logo/Brand**: "BM" text in gradient style (top left)
- **Document Title**: Inter Bold, Primary Orange
- **Subtitle**: Inter Regular, Muted Text
- **Header Height**: ~25mm

### Footer Design
- **Contact Info**: benjaminmlynek.com | your-email@domain.com
- **Page Numbers**: Right aligned
- **Footer Height**: ~15mm

## Design Elements

### Gradient Text Effect
- **Primary Gradient**: From Primary Orange (#FF6600) to Orange-400 (#FB923C)
- **Usage**: Main headings, logo text, important callouts

### Cards/Boxes
- **Background**: Card Background (#151414)
- **Border**: 1pt solid Border color (#1E2937)
- **Border Radius**: 6pt (equivalent to 0.5rem)
- **Padding**: 12pt
- **Drop Shadow**: Subtle dark shadow (optional)

### Icons
- **Style**: Line icons (similar to Lucide icons used on website)
- **Color**: Primary Orange or Foreground Light
- **Size**: 16-20pt for inline, 24pt for section headers

## Print Considerations

- Use dark backgrounds sparingly (ink-heavy)
- Ensure text has sufficient contrast
- Test print on both color and black & white printers
- Consider a "printer-friendly" version with light backgrounds
