# Guides and Downloads

This directory contains PDF guides for clients preparing for recording sessions, mixing, and mastering.

## Expected Files

Place the following PDF files in this directory:

1. **home-recording-guide.pdf** - Guide to Home Recording
   - Everything clients need to know to get professional results from their home studio setup

2. **preparing-files-for-mixing.pdf** - Preparing Files for Mixing  
   - Essential guidelines for organizing and preparing multitrack files for professional mixing

3. **preparing-files-for-mastering.pdf** - Preparing Files for Mastering
   - How to prepare final mixes for the mastering process to achieve the best possible results

4. **studio-session-preparation.pdf** - Studio Session Preparation
   - Complete checklist and guidelines for preparing for professional recording sessions

## Notes

- Files should be named exactly as listed above for the download links to work properly
- PDFs will be served directly from this public directory
- Consider keeping file sizes reasonable for web delivery
- You can update the guide descriptions in `src/pages/Guides.tsx` as needed
